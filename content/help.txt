=== DATABASE COMMANDS HELP ===

DB - database operations:
  DB CREATE [dbName]      : Create a new text database with the given name
  DB CREATEBIN [dbName]   : Create a new binary database with the given name (not implemented)
  DB OPEN [dbName]        : Open an existing database
  DB SHOW [dbName]        : Display information about the current database
  DB DELETE [dbName]      : Delete the specified database (irreversible!)

ROW - table row operations:
  ROW ADD                 : Add a new row to the current table
  ROW REWRITE [rowIndex]  : Rewrite the specified row
  ROW DELETE [rowIndex]   : Delete the specified row

COL - table column operations:
  COL ADD [nameOfColumn]      : Add a new column to the current table
  COL Edit [nameOfColumn]     : Edit the specified column (not implemented)
  COL DELETE [nameOfColumn]   : Delete the specified column (not implemented)

SORT - sorting operations:
  SORT ASC [nameOfColumn]  : Sort all rows by the specified column in ascending order (not implemented)
  SORT DESC [nameOfColumn] : Sort all rows by the specified column in descending order (not implemented)

DATA - table element operations:
  DATA EDIT [nameOfColumn] [rowIndex] [newValue] : Edit the specified element in the table (not implemented)
  DATA SHOW [nameOfColumn] [rowIndex] [newValue] : Display the value of the specified element (not implemented)

HELP - display this help information with all available commands

Notes:
- Commands marked as "not implemented" are currently unavailable.
- All commands require an open database (use DB OPEN first).
- You cannot edit columns if there are already rows in the database.
